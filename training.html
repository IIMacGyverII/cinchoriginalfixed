    Intended Audience: Installation Technicians, Service Technicians, Project Engineers.


    Purpose: Provide a complete overview of the Protector System and all of the available peripherals; learn installation best practices and how to program a complete IDS and I24 system.


    Course Objectives: After completing this course, attendees will be able to:

    Recognize system components.

    Learn proper installation procedures.

    Understand system programming.

    Test an installed system.

    Understand troubleshooting procedures.

    Train end users on system operation.


    Course Time and Delivery: This hands on course consist of 8 hours classroom time working with live equipment Course is a mix of PowerPoint presentation and hands on exercises.
