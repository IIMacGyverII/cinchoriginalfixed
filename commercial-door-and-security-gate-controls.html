Commercial Door and Security Gate Controls

Door and Security Gate control is more than opening a door or gate by button. Door and Security Gate Control is the first line of defense against entry breach. Make sure you are in control with the CeLAN Door-EZ Module.  The Door-EZ is a microprocessor based door controller system for stand-alone,  multi-door/gate and ideal for Sally Port systems when used with a CINCH control panel. Log entry and exit data, up to 260 million events with on-board, removable SD card. As with all CINCH systems products The CeLAN Door-EZ is fully AES Encrypted End-To-End to provide the highest security for doors and security gates.
